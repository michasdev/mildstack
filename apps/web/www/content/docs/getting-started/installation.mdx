## Install Once

MildStack is installed once and includes:

- MildStack Desktop App
- `mildstack` CLI
- Bundled MildStack Core runtime

<Callout type="info" title="Single installation model">
  You do not install CLI and Desktop separately. Installing the Desktop App provides the CLI and bundled runtime.
</Callout>

## Choose an Installation Method

<div className="fd-steps">

<div className="fd-step">

### macOS

Install with Homebrew (recommended):

```bash
brew tap michasdev/tap
brew install mildstack --cask
```

Or install without Homebrew:

1. Open [/download](/download) and download the macOS installer.
2. Or download the latest macOS installer from [GitHub releases](https://github.com/michasdev/mildstack/releases/latest).
3. Run the installer and open the app.

<Callout type="warn" title="Common macOS first-run issue">
  If macOS blocks the app after manual installation (and in some Homebrew installs), go to <strong>System Settings &gt; Privacy &amp; Security</strong>, scroll to the <strong>Security</strong> section, and click <strong>Open Anyway</strong>. Then open the app again.
</Callout>

</div>

## Verify the Installation

Start MildStack in the background:

```bash
mildstack start --detach
```

Then confirm the running instance:

```bash
mildstack instances
```

<Callout type="info" title="Default local endpoint">
  By default, MildStack starts on `http://localhost:4566`.
</Callout>

When you finish testing:

```bash
mildstack stop --all
```

## Update MildStack

### macOS (Homebrew)

```bash
brew update
brew upgrade mildstack
```

### macOS (without Homebrew)

Auto-updates do not work on macOS. To update, uninstall your current version and install the newest version from [/download](/download) or [GitHub releases](https://github.com/michasdev/mildstack/releases/latest).

### Windows and Linux

MildStack includes auto-update support on Windows and Linux.

To check manually for updates:

1. Open the Desktop App.
2. Go to **Settings**.
3. Click **Check for updates**.

<Callout type="warn" title="Common macOS first-run issue">
  If macOS blocks the app after manual installation (and in some Homebrew installs), go to <strong>System Settings &gt; Privacy &amp; Security</strong>, scroll to the <strong>Security</strong> section, and click <strong>Open Anyway</strong>. Then open the app again.
</Callout>
