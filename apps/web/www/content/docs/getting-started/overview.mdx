## What MildStack Is

MildStack is a local AWS emulator for development and testing on your machine. It exposes local endpoints that work with the AWS CLI and AWS SDKs for supported operations.

## What This Section Covers

This getting-started section covers installation, first-run usage, and common troubleshooting for local workflows.

## Current Service Coverage

The current documentation covers:

- Amazon S3
- Amazon DynamoDB
- Amazon SQS
- Amazon SNS
