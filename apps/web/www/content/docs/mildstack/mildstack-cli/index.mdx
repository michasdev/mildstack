The MildStack CLI is the local control surface for MildStack.
Use it to start a runtime, inspect the current snapshot, and clean up instances when you're done.
This section gives the high-level path; the linked pages cover the exact syntax and behavior.

<Callout type="info" title="Installed with Desktop">
  Installing MildStack Desktop installs the `mildstack` CLI and the packaged MildStack Core runtime.
</Callout>

## Quick Start

```bash
mildstack start
mildstack instances
```

If you do not pass a port, `start` tries `4566`, then `4567`, then `4568`.
