MildStack Desktop is the GUI companion to the `mildstack` CLI.
It manages instances through CLI-backed IPC handlers and provides service explorers for local AWS workflows.

<Callout type="info" title="Runtime requirement">
  Service navigation is disabled when the selected instance is not `running`, and service IPC handlers validate the selected instance before executing operations.
</Callout>

## Quick Start

1. Start an instance:
   ```bash
   mildstack start
   ```
2. Open the Desktop app.
3. Go to **Instances** and select a running instance.
4. Open any enabled service browser from the sidebar.

## Instances

- Lists runtime instances from `mildstack instances --json`.
- Creates or starts an instance on a selected port.
- Stops or deletes instances from the table actions.
- Selects the active instance used by service IPC requests.

## S3 Browser

- List/create/delete buckets.
- Browse object trees by prefix with client-side search and sorting.
- Upload files (drag and drop or file picker) and create folder placeholders.
- Download objects, preview supported formats, and copy S3 URIs.
- Bulk delete selected objects.

## DynamoDB Browser

- List/create/delete tables.
- View table overview, key schema, attribute definitions, and index metadata.
- Browse items with local filtering and sortable columns.
- Run advanced `Scan` and `Query` flows with filter expressions and index selection.
- Create/edit items in JSON and delete items (single or bulk).

## SQS Browser

- List/create/purge/delete queues.
- Open queue details for messages, send form, and queue attributes.
- Poll messages and delete individual messages.
- Send standard or FIFO messages (including FIFO fields).
- Show DLQ indicators when redrive metadata is present.

## SNS Browser

SNS includes four top-level sections:

- **Topics**: create/delete topics, edit attributes, publish and publish-batch, manage tags, permissions, and data protection policy.
- **Subscriptions**: list/create/confirm/delete subscriptions, inspect and update subscription attributes.
- **Platform Apps**: create/delete platform applications, edit app attributes, create/delete endpoints.
- **SMS Sandbox**: sandbox status, SMS account attributes, sandbox phone number create/verify/delete, opt-in/out checks.

## Settings

- Configure the CLI executable path used by desktop IPC.
- Save/reset/test the configured CLI path.
- Check for desktop updates manually.
- Trigger update installation when an update is available.

## Known Limits

<Callout type="warn" title="Some UI surfaces are still in progress">
  Sidebar entries `Lambda`, `SSM`, and `EventBridge` are marked as coming soon.
  In S3 bucket details, `Permissions`, `CORS`, and `Metrics` tabs are disabled and marked as coming soon.
  SQS DLQ redrive currently shows a UI flow and toasts, but does not execute a backend redrive operation yet.
</Callout>
