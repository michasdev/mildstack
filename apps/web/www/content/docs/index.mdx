## Welcome to MildStack

MildStack is a local-first AWS emulator for day-to-day development. It runs on your machine, keeps state locally, and works with the standard AWS CLI and AWS SDKs.

<Callout type="info" title="Default endpoint">
  In examples, this documentation uses `http://localhost:4566`, which is the first port checked by `mildstack start`.
</Callout>

## Start in 3 Steps

```bash
# 1) Install once (Desktop App + CLI + bundled runtime)
# 2) Start the runtime
mildstack start --detach

# 3) Call AWS APIs against the local endpoint
aws s3 mb s3://my-bucket \
  --endpoint-url http://localhost:4566
```

## CLI Commands

The current CLI command set is:

- `mildstack start [port]`
- `mildstack instances`
- `mildstack status` (alias of `instances`)
- `mildstack stop [port]` or `mildstack stop --all`
- `mildstack delete [port]` or `mildstack delete --all`
